pallet.crate.mongodb.mms documentation

Install MongoDB MMS See: http://mms.mongodb.com/help/monitoring/

server-spec

fn

[{:keys [instance-id], :as settings}]

Return a server spec for MongoDB. Keys under the :config settings will be written to the mongo .conf file. To use a replica set, specify a replica set name in the :replSet config key. Specify :arbiter true for an arbiter node (which is distinguished by the arbiter role.)

settings

fn

[{:keys [version config], :as settings} {:keys [instance-id], :as options}]

Build the configuration settings by merging the user supplied ones with the OS-related install settings and the default config settings for MongoDB
