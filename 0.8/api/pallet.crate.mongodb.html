server-spec

fn

[{:keys [arbiter instance-id], :as settings}]

Return a server spec for MongoDB. Keys under the :config settings will be written to the mongo .conf file. To use a replica set, specify a replica set name in the :replSet config key. Specify :arbiter true for an arbiter node (which is distinguished by the arbiter role.)
